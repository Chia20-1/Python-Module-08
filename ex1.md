# PM08 - Exercise 1 Guide

This guide is for `ex1`, the "Loading Programs" exercise. The main lesson is how Python projects describe and install external packages using two common approaches:

- `requirements.txt` for `pip`
- `pyproject.toml` for Poetry

## Segment 1: Understand The Subject Requirement

### What the exercise is asking for

The subject wants you to create a Python data analysis program named:

```text
ex1/loading.py
```

The program should simulate and analyze "Matrix data", then create a simple visualization image.

The important point is not only the analysis. The exercise is also testing whether you understand how to manage project dependencies with both `pip` and Poetry.

### Files you must submit

Inside `ex1/`, the subject asks for these files:

```text
loading.py
requirements.txt
pyproject.toml
```

### Authorized modules

The authorized modules are:

```text
pandas
requests
matplotlib
numpy
sys
importlib
```

`requests` is optional. You only need it if you decide to fetch real data from an API. The subject says simulated data is enough.

### Required behavior

Your program must:

- Use `numpy` to generate the dataset.
- Use `pandas` to manipulate or analyze the dataset.
- Use `matplotlib` to create a visualization.
- Create a saved image, for example `matrix_analysis.png`.
- Handle missing dependencies gracefully.
- Include a comparison function that shows installed package versions.
- Show the difference between `pip` and Poetry in the program output.
- Provide dependency files for both package-management methods:
  - `requirements.txt`
  - `pyproject.toml`

### Important detail: numpy must create the data

The subject says `numpy` must be the source of your dataset.

That means you should avoid doing this:

```python
data = [1, 2, 3, 4, 5]
```

You should also avoid using normal `range()` as the main source of the dataset.

Instead, use `numpy`, for example:

```python
rng = np.random.default_rng(seed=42)
signal_strength = rng.normal(loc=70, scale=10, size=1000)
latency = rng.integers(low=10, high=200, size=1000)
```

Then you can pass those arrays into a `pandas.DataFrame`.

### What "handle missing dependencies gracefully" means

Do not let the program crash with a long `ModuleNotFoundError` traceback.

For example, avoid importing external packages at the very top like this:

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
```

If one package is missing, the program stops immediately before you can print a helpful message.

Instead, use `importlib` to check packages first. If something is missing, print instructions such as:

```text
Missing dependency: pandas

Install with pip:
python -m pip install -r requirements.txt

Or install with Poetry:
poetry install
poetry run python loading.py
```

### What the pip file should do

`requirements.txt` is the simple `pip` dependency list.

Example:

```text
numpy>=1.25.0
pandas>=2.1.0
matplotlib>=3.7.0
```

If you use `requests`, add:

```text
requests>=2.31.0
```

Install it with:

```powershell
python -m pip install -r requirements.txt
```

### What the Poetry file should do

`pyproject.toml` is the Poetry project configuration file. It stores project information and dependencies.

Example shape:

```toml
[tool.poetry]
name = "loading-programs"
version = "0.1.0"
description = "Matrix-themed dependency management exercise"
authors = ["Your Name"]
package-mode = false

[tool.poetry.dependencies]
python = ">=3.10,<4.0"
numpy = "^1.25.0"
pandas = "^2.1.0"
matplotlib = "^3.7.0"

[build-system]
requires = ["poetry-core"]
build-backend = "poetry.core.masonry.api"
```

If you use `requests`, add it under `[tool.poetry.dependencies]`.

Install and run with:

```powershell
poetry install
poetry run python loading.py
```

### How to explain pip vs Poetry in your output

Your program can print a short comparison:

```text
Dependency management comparison:
- pip reads requirements.txt and installs packages into the active environment.
- Poetry reads pyproject.toml, creates or uses an isolated environment, and manages dependency resolution.
```

That satisfies the requirement to show the difference through the program output.

## Segment 2: Learn How To Do This Exercise

### Recommended file structure

By the end, `ex1/` should look like this:

```text
ex1/
  loading.py
  requirements.txt
  pyproject.toml
  matrix_analysis.png
```

`matrix_analysis.png` is generated by the program, so it may not need to be submitted unless your evaluator expects generated output.

### Step 1: Plan the program functions

A clean way to structure `loading.py` is:

```text
main()
  check_dependencies()
  show_dependency_management_comparison()
  simulate_matrix_data()
  analyze_matrix_data()
  generate_visualization()
```

Suggested responsibilities:

- `check_dependencies()` checks `numpy`, `pandas`, and `matplotlib`.
- `show_dependency_management_comparison()` prints the pip vs Poetry explanation.
- `simulate_matrix_data()` uses `numpy` to generate 1000 fake data points.
- `analyze_matrix_data()` uses `pandas` to calculate summary values.
- `generate_visualization()` uses `matplotlib` and saves `matrix_analysis.png`.

### Step 2: Check dependencies before using them

Use `importlib.import_module()` to import packages by name.

Example idea:

```python
module = importlib.import_module("numpy")
```

Store the loaded modules in a dictionary so you can use them later:

```python
modules["numpy"] = module
```

You can read the version like this:

```python
version = getattr(module, "__version__", "unknown")
```

If an import fails, catch `ImportError`, print a helpful message, and stop the program cleanly.

### Step 3: Generate Matrix data with numpy

Use `numpy` as the source of all data columns.

Example columns you can create:

- `signal_strength`
- `latency_ms`
- `anomaly_score`
- `energy_usage`

Example idea:

```python
rng = np.random.default_rng(seed=42)

data = {
    "signal_strength": rng.normal(70, 10, 1000),
    "latency_ms": rng.integers(10, 200, 1000),
    "anomaly_score": rng.random(1000),
    "energy_usage": rng.normal(50, 8, 1000),
}
```

Then create a DataFrame:

```python
df = pd.DataFrame(data)
```

### Step 4: Analyze the data with pandas

Keep the analysis simple. The subject does not ask for complex statistics.

Good examples:

```python
summary = df.describe()
average_latency = df["latency_ms"].mean()
high_anomalies = df[df["anomaly_score"] > 0.95]
```

You can print:

- Number of data points processed.
- Average latency.
- Average signal strength.
- Number of suspicious or high-anomaly rows.

### Step 5: Create a visualization with matplotlib

Create one simple plot and save it.

Good options:

- Histogram of `signal_strength`
- Scatter plot of `latency_ms` vs `anomaly_score`
- Line plot of the first 100 generated values

Example idea:

```python
plt.figure(figsize=(8, 5))
plt.hist(df["signal_strength"], bins=30)
plt.title("Matrix Signal Strength Distribution")
plt.xlabel("Signal Strength")
plt.ylabel("Frequency")
plt.tight_layout()
plt.savefig("matrix_analysis.png")
plt.close()
```
